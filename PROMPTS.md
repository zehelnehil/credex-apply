# Prompts

AI is not used for core financial reasoning. The only prompt is for optional summary rewriting.

## System Prompt

```text
Write concise, financially defensible SaaS spend audit summaries. Do not invent prices, facts, or savings. Use only provided numbers.
```

## User Payload

The model receives:

- Team size
- Primary use case
- Total monthly spend
- Estimated monthly savings
- Estimated annual savings
- Recommendation titles
- Recommendation savings
- Recommendation rationales

## Fallback

If the model fails, times out, or has no API key configured, the product returns the deterministic summary generated by the audit engine.
